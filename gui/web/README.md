# JARVIS HUD UI

A futuristic JARVIS-style interface built with Vite + Vanilla JS, featuring:

- Circular reactor HUD inspired by cinematic JARVIS ring designs
- Continuous ring motion with glowing core and rotating segmented arcs
- Automatic system mode cycle:
	- `SYSTEM OFFLINE` → glow and ring motion pause
	- `SYSTEM ONLINE` → glow returns and circular motion resumes
- Lightweight single-page architecture

## Setup

```bash
yarn install
```

## Run (Development)

```bash
yarn dev
```

## Build (Production)

```bash
yarn build
```

## Preview Build

```bash
yarn preview
```
